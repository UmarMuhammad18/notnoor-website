# N0tN0Or - Luxury Modest Fashion

A highly customized, single-page application (SPA) luxury e-commerce landing page built specifically for N0tN0Or, a Hijabi Ready-to-Wear (RTW) brand offering worldwide delivery and custom sizing.

## 🌟 Features

- **Luxury UI/UX:** Built with high-end aesthetics, smooth reveal animations, marquee banners, and a dark-theme luxury design.
- **Enhanced Product Modals:** Two-column product detail modals featuring high-resolution images, size selectors, descriptions, and direct **WhatsApp Order** integration.
- **Multi-Currency Support:** Seamlessly switch between NGN (₦), USD ($), GBP (£), and EUR (€) in the navigation bar to support worldwide shoppers.
- **Community & Newsletter:** Integrated email capture mechanism to build an audience.
- **Dedicated FAQ:** Interactive accordion-style FAQ section addressing custom fabrics, shipping, and return policies.
- **Owner Admin Dashboard:** A built-in, password-protected dashboard (accessible via `<brandname>`, `<admin>`, or `<noor>` login) allowing the owner to:
  - Add/Edit/Delete products in real-time.
  - Manage lookbook items and customer testimonials.
  - Track customer orders and update shipping statuses.
  - Push real-time top-bar announcements to visitors.

## 🛠 Tech Stack

- **HTML5** & **CSS3**: Pure vanilla CSS variables and animations for lightning-fast loading speeds (No external heavy frameworks).
- **JavaScript (ES6)**: State management and DOM manipulation powered by pure vanilla JavaScript. Uses browser `localStorage` as a lightweight built-in database to persist product edits and admin settings locally.

## 🚀 How to Run

1. Clone the repository natively or simply download the source files.
2. Open `n0tn0or_full.html` in any modern web browser (Chrome, Safari, Firefox, Edge).
3. The site is fully functional offline due to its unique `localStorage` backend architecture.

## 🔐 Admin Access
To access the dashboard:
1. Click the **"⚙ Owner"** button in the top navigation.
2. Enter the username: `notnoor`
3. Enter the default password (provided securely to owner).
4. Update products, add sales banners, and manage reviews effortlessly.
